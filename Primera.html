VALORACIONES ÁCIDO-BASE



La valoración ácido-base es el proceso mediante el cual se determina la concentración
de un analito que funciona como ácido o base en una solución mediante la reacción de
neutralización con una base o un ácido de concentración conocida (titulante o solución
patrón).
La técnica que se utiliza para llevar a cabo este proceso se denomina volumetría, ya que
hace referencia a la medida del volumen de las disoluciones empleadas, que nos permiten
calcular la concentración del analito buscada.
